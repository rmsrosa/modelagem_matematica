Validação cruzada¶

  • O objetivo aqui no apenas achar os parmetros do modelo que melhor se ajustam aos dados, mas verificar se ele um bom modelo para fazer previses/predies.

  • A ideia separar parte dos dados para o ajuste (treino) e o restante para validao (teste).

  • Separaes tpicas so da ordem de 75% dos dados para o ajuste e 25% para teste.

  • Compara-se o RMS do ajuste com o RMS do teste:

    • Se o RMS do teste estiver prximo do RMS do ajuste, considera-se que o modelo tem boa capacidade de previso.

    • Se o RMS do teste for muito maior, o modelo no tem boa capacidade de previso.

Validação cruzada com dobras (k-fold cross-validation)¶

  • Nesse caso, a ideia fazer vrias validaes cruzadas.

  • Divide-se os dados em $J$ partes iguais (chamadas dobras, ou folds), digamos, $J=5$ ou $10$.

  • Em cada uma de $J$ etapas, seleciona-se 1 das dobras para teste e as outras $J-1$ para o ajuste.

  • Ao final, temos $J$ conjuntos $\beta^{j}$ de parmetros, cada um como melhor ajuste baseado nos seus dados de treino, assim como $J$ valores de RMS do ajuste e $J$ de RMS do teste.

  • Se os valores dos parmetros, nas diferentes dobras, estiverem prximos uns dos outros, isso um indicativo da estabilidade do modelo.

  • Ao final, se o modelo for estvel, podemos selecionar os parmetros de vrias formas:

    • Fazendo um novo ajuste com todos os dados;

    • Tomando uma mdia, entre as vrias dobras, de cada parmetro; ou

    • Selecionado uma ou tomando uma mdia entre algumas dobras dentre as mais estveis e com melhor qualidade preditiva.

  • Considera-se o erro RMS da validao cruzada como a mdia quadrdica dos erros de cada dobra

$$ \epsilon = \sqrt{\frac{1}{J}\sum_j \epsilon_j^2}. $$

Exemplo sintético¶

Neste exemplo sinttico, vamos

  • Construir dados sintticos a partir de perturbaes aleatrias em torno de uma determinada funo.

  • Perturbaes uniformemente distribudas entre $\pm 0.3$ na abscissa.

  • Perturbaes uniformemente distribudas entre $\pm 1$ na ordenada.

  • Em seguida, usaremos mnimos quadrados para ajustar um polinmio de grau trs.

  • Posteriormente, mnimos quadrados para ajustar polinmios de diversos graus.

Função para cálculo das medidas de qualidade do modelo¶

function info_ajuste(dados_x, dados_y, model_y, k)
    N = length(dados_x)
    y_mean = mean(dados_y)
    residuos = model_y - dados_y
    ss = norm(residuos)^2
    rms = sqrt(ss/N)
    ss_y = norm(dados_y)^ 2
    rms_y = sqrt(ss_y/N)
    ss_rel = ss/ss_y
    rms_rel = sqrt(ss_rel)
    ss_tot = N*var(dados_y)
    ss_reg = norm(model_y .- y_mean)^2
    r_sq = ss_reg/ss_tot
    r_sq_aj = 1 - (1 - r_sq)*(N-1)/(N-k)
    
    return (
        residuos=residuos, rms=rms, rms_rel=rms_rel, ss=ss, ss_rel=ss_rel,
        r_sq=r_sq, r_sq_aj=r_sq_aj
    )
end
