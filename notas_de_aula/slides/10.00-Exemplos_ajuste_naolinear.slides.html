Exemplos de ajuste não-linear de parâmetros¶

  • H vrios pacotes de otimizao disponveis no julia

  • JuliaNLSolvers/LsqFit.jl: implementao do algoritmo de Levenberg-Maquardt especificamente para mnimos quadrados no-linear.

  • JuliaNLSolvers/Optim.jl: implementao de vrios algoritmos de otimizao, com qualquer funo-objetivo, no necessariamente mnimos quadrados.

  • JuliaNLSolvers/LineSearches.jl: implementao de vrios algoritmos do tipo busca linear (linesearch) (como o gradiente descendente, etc.).

  • jump-dev/JuMP.jl: para processos de modelagem mais complexos, com uma srie de macros para construir um modelo para a otimizao, e com interface para vrios pacotes (em julia ou no) de otimizao.

  • Aqui, vamos usar apenas o LsqFit e o Optim.

Exemplo Michaelis-Menten de reação enzimática¶

  • Michaelis-Menten um modelo para a taxa de reao enzimtica (velocidade) $\nu$, dado por $$ v = \frac{\nu_m t}{K_M + t} $$

  • Onde $t$ a varivel temporal, $\nu_m$ uma taxa mxima de velodidade, e $K_M$ um parmetro de saturao da reao.

In [2]:
function model(t, p)
    _m = p[1]
    K_M = p[2]
    v = (_m .* t) ./ (K_M .+ t)
    return v
end

_m = 0.3
K_M = 0.5
p = [_m K_M]
nothing

Obtendo uma amostra de dados¶

  • Perturbamos os dados exatas para simular imprecises na coleta de dados

data_t = [0.03, 0.15, 0.3, 0.6, 1.3, 2.5, 3.5]
data_v = model(data_t, p) .+ 0.05*(rand(MersenneTwister(321), length(data_t)) .- 0.5)
plot(0:0.1:4, t -> model(t, p), label="taxa de reao", legend=:bottomright)
plot!(data_t, data_v, seriestype=:scatter, label="amostra com rudo")

Ajuste dos dados via Optim¶

  • Aqui, usamos o mtodo de gradiente conjugado do Optim.

  • No Optim, precisamos explicitar a funo custo.

  • Lembremos que queremos ajustar os parmetros, ou seja, precisamos minimizar o custo em relao aos parmetros.

In [6]:
custo(p) = sum(abs2,data_v - model(data_t,p))

Dados espúrios (outliers)¶

  • comum, por dificuldades no processo de amostragem ou de tratamento dos dados, termos dados "esprios", i.e. claramente fora do padro.

  • So denominados de outliers.

  • Podemos tentar filtrar os dados esprios da amostra usando tcnicas de estatstica.

  • E/Ou podemos reduzir o impacto dos dados esprios com funes de custo ligeiramente modificadas.
